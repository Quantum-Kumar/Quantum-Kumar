## 📂 Featured Projects

### ⚛️ Quantum Hardware & Simulation Projects
Here are some of my recent hands-on projects involving various quantum processors and simulation frameworks. You can find the code for these in my [Quantum-Coder repository](https://github.com/Quantum-Kumar/Quantum-Coder/tree/main/Quantum_Projects).

- **Neutral Atom Characterization (QuEra Aquila):** Executed Analog Hamiltonian Simulation (AHS) on QuEra’s neutral-atom processor via Amazon Braket. Measured second-order correlation functions to study spatial correlations and Rydberg blockade. Built Python pipelines to process results and investigate noise and calibration drift.
- **Hamiltonian Learning & Adaptive Calibration (IAS Framework):** Developed Iterative Adaptive Spectroscopy routines using Python, QuTiP, and Qiskit Dynamics to extract Hamiltonian parameters under noise. Created automated workflows to estimate spectral leakage and optimize measurements.
- **Rigetti Hardware Experiments (Cepheus-1-108Q):** Ran Rabi oscillation and Ramsey interferometry experiments on Rigetti's superconducting hardware via Amazon Braket. Prepared Bell states and analyzed hardware noise against simulation predictions.
- **Ramsey Characterization (IBM Quantum):** Implemented Ramsey pulse sequences on IBM backends using Qiskit and OpenPulse to estimate T1/T2 coherence times and monitor calibration drift. Modeled driven qubit dynamics to study gate errors.
- **Quantum Error Correction Simulations:** Simulated Surface Code, Steane, and Toric Code circuits using STIM and Qiskit. Built syndrome extraction and decoding workflows with Minimum Weight Perfect Matching (MWPM) to study fault-tolerance thresholds and logical error rates.
